# PharmaTrack
SE 4347.001 - Database Systems Spring 2026

## Setup

1. Install and start XAMPP
2. In XAMPP, start Apache and MySQL. Both should turn green
3. Go to http://localhost/phpmyadmin in your browser
4. Create a database named pharmatrack
5. Open the database, then go to SQL
6. Run Create.sql
7. Run Load.sql
8. Copy this project folder into C:\xampp\htdocs\pharmatrack
9. Open http://localhost/pharmatrack/homepage.php
10. You should now be able to demo PharmaTrack
